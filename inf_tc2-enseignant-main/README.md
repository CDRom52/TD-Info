Ce répertoire contient `la version enseignant` des sujets de TD pour l'enseignement `INF-TC2 Conception et Programmation Objet` et n'est visible que par les enseignants. 

A chaque séance correspond un dossier contenant le corrigé sous forme d'un fichier markdown (d'extension .md) ainsi que le code Python.

Les versions étudiant sont disponibles [à cette adresse](https://gitlab.ec-lyon.fr/edelland/inf_tc2_2025-2026-S5), et les supports de cours sur [Moodle](https://pedagogie1.ec-lyon.fr/course/view.php?id=2013).
